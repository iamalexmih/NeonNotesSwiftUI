# app Neon Notes.

- Приложение для ведения заметок.
- 3 экран:
    - основной экран со списком хранимых заметок.
    - экран создания заметки.
    - экран редактирования заметки.
- Стэк: 
    - SwiftUI.
    - MVVM.
    - Combine.
    - Core Data.
- Удаление заметок по свайпу влево.
